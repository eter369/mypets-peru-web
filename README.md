# MyPets Perú - Sitio Web

Sitio web de catálogo de cachorros con carrito, productos exóticos, sello de garantía y CTAs a WhatsApp.

## Estructura

- `index.html` — sitio completo (HTML + CSS + JS embebidos)
- `images/` — imágenes de cachorros, sellos y branding
- `brand-image.js` — script Node con sharp para aplicar branding a fotos

## Desarrollo local

```bash
npm install
npx serve .
```

Abrir `http://localhost:3000`.
